# Categories
- Possible values for selecting the default category from the config is defined below.
  - e.g. `default_category = 'AnimeEnglishTranslated'`

## Anime
- `AllAnime`
- `AnimeEnglishTranslated`
- `AnimeNonEnglishTranslated`
- `AnimeRaw`
- `AnimeMusicVideo`

## Audio
- `AllAudio`
- `AudioLossless`
- `AudioLossy`

## Literature
- `AllLiterature`
- `LitEnglishTranslated`
- `LitEnglishTranslated`
- `LitRaw`

## Live Action
- `AllLiveAction`
- `LiveEnglishTranslated`
- `LiveNonEnglishTranslated`
- `LiveIdolPromoVideo`
- `LiveRaw`

## Pictures
- `AllPictures`
- `PicGraphics`
- `PicPhotos`

## Software
- `AllSoftware`
- `SoftApplications`
- `SoftGames`
