# Keybinds
Defined below are the keybinds for all modes.

## Normal
| Key | Map |
| --- | --- |
| Enter | Confirm |
| q | Exit App |
| g | Top |
| G | Bottom |
| j | ↓ Down |
| k | ↑ Up |
| n, l, → | Next Page |
| p, h, ← | Prev Page |
| r | Reload |
| /, i | Search |
| c | Categories |
| f | Filters |
| s | Sort |
| S | Sort reversed |
| t | Themes |
| Ctrl-P | Goto page |
